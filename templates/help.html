Start here

Use the CRM in a simple order so it does not feel heavier than it needs to.

The core path is customer, quote, job, invoice. Reports, reminders, campaigns, recurring income, and advanced SMS sit around that workflow and should only be used when you actually need them.

Daily workflow

  1. Check quote requests and new customer enquiries.
  2. Book or update jobs in Jobs or Calendar.
  3. Raise invoices for completed work.
  4. Send reminders only where needed.

Weekly workflow

  1. Review reports and cashflow.
  2. Log expenses consistently.
  3. Check recurring income due soon.
  4. Take a backup before major changes.

Where to work from

  1. Use Overview for next actions only.
  2. Use Customer Library to find the record, then work from the customer page itself.
  3. Use Finance for money admin.
  4. Use Communications only after the core workflow is moving.

What each section is for

SectionUse it for
OperationsCustomers, jobs, scheduler, and the full calendar.
SalesQuote requests, quotes, the public portal, and pricing.
FinanceInvoices, reminders, expenses, recurring income, and reports.
CommunicationsOne off messaging, templates, SMS, campaigns, and segments.
SystemSettings and backups.

Best practice

  • Keep customer details accurate before doing anything else.
  • Use the customer record as the main hub for that job lifecycle.
  • Do not try to work out of Reports or Communications all day.
  • Only use campaigns and advanced SMS once the basics feel natural.
  • Use Backups before big imports, edits, or testing.
